## Components to Use

### Navigation Container

- Type: fixed top
- Background: #F9F7F3
- Border Bottom: 1px solid #E6E3DE
- Padding: 16px

### Logo

- Font: Playfair Display Bold
- Size: 24px
- Color: #1C1C1C

### Navigation Links

- Font: Inter, 18px
- Color: #1C1C1C

### Hamburger Menu

- Opens a full-screen overlay on mobile
- Menu links stacked vertically, centered

### CTA Buttons

- Primary: “Start Customizing”
    - Background: #C9A14A
    - Text: white, 16px Inter SemiBold
    - Border Radius: 8px
    - Touch target ≥48px
- Secondary: “Shop Collection”
    - Border: 2px solid #C9A14A
    - Text: #1C1C1C

## Animations / Effects

### Nav scroll behavior

- Shrinks height slightly when scrolling down
- Logo scales down to 0.9x

## Design Instructions

- Focus on **thumb-friendly spacing**
